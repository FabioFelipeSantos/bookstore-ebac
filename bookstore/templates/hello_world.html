Sobre o projeto

Projeto com o intuito de realizar uma aplicação completa em Django e fornecê-la através da plataforma Python Anywhere.

O objetivo é praticar os conceitos aprendidos no curso de Desenvolvimento Full Stack Python da Ebac.
